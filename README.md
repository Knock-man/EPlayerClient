# 易播播放器客户端
基于Qt VLC编写的视频播放器
# 开发环境
Qt5.12.9   MSVC2017_32bit  
# 运行步骤
1.打开VideoPlayer文件夹  
2.点击 VideoPlayer.pro文件运行
